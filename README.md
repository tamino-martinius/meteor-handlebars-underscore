# handlebars-underscore

Port of some underscore-js functions for handlebars.

## Compare

```
$ne
$eq
$gt
$gte
$lt
$lte
```

## Collections

```
$contains
$indexOf
$lastIndexOf
$min
$max
$first
$last
$flatten
$compact
$intersection
$difference
```

## Objects

```
$keys
$values
$pairs
$invert
$functions
$pick
$omit
$has
```
